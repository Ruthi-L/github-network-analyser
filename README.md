# github-network-analyser
A graph-based analysis tool for modeling and visualizing mutual follower relationships using Python and Pyvis.

## Core Features
* **Graph Modeling:** Uses adjacency-based mapping to identify clusters in developer ecosystems.
* **Visualization:** Implemented using **Pyvis** for interactive node-edge exploration.
* **Optimization:** Includes data caching to manage API rate limits and ensure performance.

## Tech Stack
* Python
* Pyvis / NetworkX
* GitHub API


This implementation represents the GitHub social ecosystem as an Adjacency List using networkx. The project focuses on Breadth-First traversal concepts to map user connectivity and visualize sub-graph clusters
